Bienvenue sur ce projet :'Recherche Dechotomique dans un fichier vu comme un tableau d'enregistrement de taille variable
                           non ordonné sans chevauchement'
Cours : Structure de fichiers & Données.
Réalisé par : 
	-->https://github.com/OussamaBenakmoum
Énoncé : 
	-->Donner l'Algorithme de recherche dichotomique pour un fichier TOV-C dont l'unique caractéristique est : le numéro de dernier bloc.
	La structure des blocs est comme suit:
					Tbloc = structure
					  tab : tableau[b] de caractères //contenant les enregistrements.
					  pos_libre : entier  //indice de derniere position libre dans le tableau.
					  clé1,clé2 : chaine de 10 caractères //la plus petite et grande clé dans le tableau (Le bloc)
					Fin;
	On supposera que chaque enregistrement est stocké sous forme de chaine de caractères commençant par sa taille (3 caractères),un caractère d'effacement 
	logique , puis sa clé de taille fixe (10 caractères).Le rest de caractères représentent les champs restants de l'enregistrement.
